# Installation

## Stable release

To install Container RL Env, run this command in your terminal:

```sh
uv add container-rl
```

Or if you prefer to use `pip`:

```sh
pip install container-rl
```

## From source

The source files for Container RL Env can be downloaded from the [Github repo](https://github.com/guyreading/container_rl).

You can either clone the public repository:

```sh
git clone https://github.com/guyreading/container_rl
```

Or download the [tarball](https://github.com/guyreading/container_rl/tarball/main):

```sh
curl -OJL https://github.com/guyreading/container_rl/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd container_rl
uv sync
```
